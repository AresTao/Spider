Spider
======
简介：
======
一个简单易用易扩展的网络爬虫，用户只需要实现自己的PageVisitor即可按照自己的意愿进行网页爬取。

使用方法：
=========
git clone git@github.com:AresTao/Spider.git将工程拷贝到本地

右键pom.xml

执行：maven build install即可完成项目构建

开发人员可以使用spider-v1.0.jar进行开发
